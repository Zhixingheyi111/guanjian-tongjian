# 观鉴

沿时间读懂《资治通鉴》，再把历史变成自己的判断。

这是一个 React/Vite PWA。产品主线面向不了解中国历史的读者，以“历史总览 → 时代导读 → 阶段时间线 → 核心故事 → 原文与臣光曰 → 理解检查”组织内容，而不是要求用户直接从294卷原书目录中寻找学习路径。

## 在线学习

公开版：[观鉴 · 资治通鉴学习](https://zhixingheyi111.github.io/guanjian-tongjian/)

GitHub Pages 目前作为公开备用站。Cloudflare Pages 镜像使用同一份代码与本地地图资源，完成账号授权后会发布到 `guanjian.pages.dev`，再视中国大陆实测结果决定是否接入独立域名和境内备案部署。

- iPhone / iPad：用 Safari 打开，点“分享”后选择“添加到主屏幕”。
- Android：用 Chrome 打开菜单，选择“安装应用”或“添加到主屏幕”。
- 阅读进度、理解检查、收藏和笔记保存在当前设备的浏览器中。

## 信息架构

- `通鉴`：默认展示十个历史阶段。每个阶段先学习社会背景、人物阵营、核心矛盾和兴衰章节，再进入核心故事；可切换到294卷原书目录。
- `问事`：输入现实处境，结合已收录的历史故事分析怎么判断、怎么说、怎么做。
- `处世`：按修身、识人、言语和合作分类检索方法，并回到对应故事学习完整上下文。
- `笔记`：保存用户主动收藏、摘录和写下的心得。

历史故事只有一个完整详情页。时代导读、卷目、搜索、问事和处世只是不同入口，都会指向同一个故事课；故事页同时标明其所属时代和章节。

## 时代导读

十个时代均具备统一学习结构：

1. 八分钟时代全貌与地图
2. 权力、财政、军事和社会生活
3. 关键人物、政权与阵营关系
4. 三至五项长期核心矛盾
5. 四至七个连续历史章节
6. 四至六道因果理解题
7. 核心故事与按章节收纳的延伸阅读

当前共有53个历史章节、73个关键人物/阵营、60条关系和41道理解题。时代导读和答题进度使用本地存储，刷新后保留，不会覆盖旧笔记与收藏。

## 当前内容

全书卷001至卷294已完成第一轮连续精编：每卷都可沿时间轴进入，至少包含一个关键故事、一段重点原文、自写白话讲解和一条从故事中提炼的处世方法。重要节点另收录司马光“臣光曰”原文、讲解与相应时代地图。

当前数据规模：10个历史阶段、294卷、295课、297个重要事件、17条司马光评价、14张时代地图和310条处世方法。所有295课都归入明确的时代章节，并标记为核心主线或延伸阅读。第一轮不等于每卷所有事件与全部“臣光曰”已经穷尽；后续扩写继续在同一章节和卷内增加故事，不再另造重复入口。

卷001的完整故事课是：`晋国为什么会分裂：从智伯立瑶到三家分晋`。

课程按因果顺序展示：

1. 智伯立瑶
2. 智伯索地
3. 晋阳被围
4. 韩魏倒戈、智氏覆灭
5. 前403年周天子册命三家

对应的重点原文、两段完整“臣光曰”、白话讲解、地图说明和处世方法都放回相关节点。

## 开发

```bash
npm install
npm run dev
npm run validate:data
npm run lint
npm run build
npm run build:cloudflare
```

## 部署

### GitHub Pages

```bash
npm run build:pages
```

该构建使用 `/guanjian-tongjian/` 子路径，并生成 GitHub Pages 所需的 `404.html`。

### Cloudflare Pages

Cloudflare Pages 项目使用以下设置：

- 项目名：`guanjian`
- 生产分支：`main`
- 构建命令：`npm run build:cloudflare`
- 输出目录：`dist`
- Node.js：22

`public/_redirects` 保证直接打开时代、故事和卷详情时仍进入 React Router；`public/_headers` 让 Service Worker 及时更新，同时长期缓存带哈希的静态资源。

完成 Wrangler 登录后也可以从本机直接发布：

```bash
npm run deploy:cloudflare
```

## 可选 AI

开发服务器通过 `/api/consult` 提供可选代理。前端不会保存或暴露 API key：

```bash
OPENAI_API_KEY=你的密钥 npm run dev
```

没有 `OPENAI_API_KEY` 时，“问事”会使用本地历史故事和处世方法生成参照分析。

## 数据原则

- 原文、事实、白话与现代方法分层保存。
- 每条原文、臣光曰和地图必须记录来源。
- 历史事件是时间线节点；故事课是学习单位；卷目是原书索引。
- 不用空白课程制造“已经完成”的假象；数据校验保证294卷不能退回占位状态。
